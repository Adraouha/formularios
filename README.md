# formularios
Formulario de contacto

# Instrucciones

Crear un formulario de consulta para tu empresa con los siguientes campos:

- Nombre
- Apellido (**required**)
- Edad  
  - Mínimo: 18  
  - Máximo: 120  
  - Mostrar estos valores en el *placeholder*
- Email (**required**)
- Fecha
- Asunto (**required**)
- Comentario  
  - Máxima longitud: 150 caracteres

El formulario debe enviarse mediante el método **GET** a la página `recibido.html`, la cual debe mostrar un mensaje del tipo:

> “¡Mensaje recibido!”

Verificar que los datos enviados aparezcan en la barra de navegación (URL).

## Estilo CSS

Una vez que el formulario funcione correctamente:

- Aplicar estilos con CSS
- Utilizar colores personalizados
- Definir una tipografía adecuada
- Mejorar la distribución de los elementos
- Ajustar márgenes, espaciados y alineación
- Diseñar una interfaz visualmente agradable
